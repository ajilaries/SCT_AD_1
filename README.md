# SCT_AD_1
As part of my Android Development Internship, my first task is to develop a basic Calculator Application using Kotlin.

The application is designed to perform fundamental arithmetic operations such as addition, subtraction, multiplication, and division. The goal of this task is to understand core Android development concepts including user interface design, event handling, and implementing logical operations in Kotlin.

Through this project, I am gaining hands-on experience in:

Building interactive UI using XML
Handling user inputs through button clicks
Applying Kotlin programming logic for calculations
Managing app state and displaying results dynamically

This task serves as a strong foundation for developing more complex Android applications in the future.
